# 455. Assign Cookies

https://leetcode.com/problems/assign-cookies/

## Solution:

```python
class Solution:
    def findContentChildren(self, g: List[int], s: List[int]) -> int:
        g.sort()
        s.sort()
        i, j = 0, 0
        while i < len(g) and j < len(s):
            if s[j] < g[i]:
                j += 1
            else:
                j += 1
                i += 1
        return i
```

## Analysis:

Let $N$, $M$ be the number of elements in `g`, `s` respectively.

### Time Complexity:

$O(N\log N + M\log M)$

### Space Complexity

$O(1)$ + that of `list.sort()`.
