# 1299. Replace Elements with Greatest Element on Right Side

## Solution:

```python
class Solution:
    def replaceElements(self, arr: List[int]) -> List[int]:
        cur_max = -1
        next_max = None
        for i in range(len(arr) - 1, -1, -1):
            next_max = max(cur_max, arr[i])
            arr[i] = cur_max
            cur_max = next_max
        return arr
```

## Analysis:

Let $N$ be the number of elements in `arr`.

### Time Complexity:

$O(N)$

### Space Complexity

$O(1)$
